# @amlplugins/google-adsense

Native `@googleapis/adsense` SDK re-exported under the `@amlplugins` namespace.

> AdSense Management.

## Install

```bash
npm install @amlplugins/google-adsense
```

## Use

```ts
import { /* named exports from @googleapis/adsense */ } from "@amlplugins/google-adsense";
// or
import sdk from "@amlplugins/google-adsense";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
